#  AGROAI – Plant Disease Prediction System

##  Overview
AGROAI is a full-stack web application that predicts plant diseases using a Deep learning model.

The system consists of:
- React + TypeScript frontend
- Python (Flask/FastAPI) backend
- LightPLDnet



## 🛠 Tech Stack

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

### Backend
- Python
- Flask (or FastAPI if used)
- LightPLDnet



## 📂 Project Structure

AGROAI/
│
├── frontend/
├── backend/
│   └── app.py



## ⚙ Installation

### Clone Repository
git clone https://github.com/Manav9334/AGROAI.git


### Frontend Setup
cd frontend
npm install
npm run dev


### Backend Setup
cd backend
pip install -r requirements.txt
python app.py


## 🚀 Features
- Plant image upload
- Disease prediction
- Responsive UI
- Backend ML integration
